# LIFRANUM
Automated analyse of natively digital French literature

## Requirements
```bash
spacy >= 3.0.1
nltk >= 3.5
gensim >= 4.1.0
Mallet 2.0 binary file, which path should be updated in the code/other/config.py file
```
